Security Architecture

Your keys. Your container. Your control.

Every user operates in a fully isolated environment. No shared access. No exceptions.

🐳
Private Docker Container

Each user gets a dedicated, isolated Docker container with their own copy of the system. Your data, your keys, your logic — completely separate.

🔑
Read + Trade Only

We only request Read and Instant Trade API permissions. Withdrawal and transfer are never requested — we strongly recommend disabling them on Binance too.

🛡️
Zero Shared Access

No user can see another user's data, trades, or configurations. The MiniShell gateway ensures all sessions are authenticated and fully isolated.

☁️
Cloud-Powered Infrastructure

Running on Azure. Agent communication via Azure Service Bus. You control your container. We control nothing about your funds.
